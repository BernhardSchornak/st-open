OnLine-Hilfe (OLH)


Beide Dialoge verwenden die Grösse der Original-MLEs. So haben Sie sofort ein Bild davon, wie der eingegebene Text später einmal im Hilfe-Dialog aussehen wird.

Hilfetexte

Den Dialog zur Eingabe und Übersetzung von Hilfetexten erreichen Sie über Tools - OnLine-Hilfe. Alle Eingaben werden automatisch in das OLH-Feld geschrieben, sobald das gerade bearbeitete Dialogelement den Eingabefokus verliert.

Sprachauswahl

Wählen Sie mit den <<- bzw. >>-Knöpfen die gewünschte(n) Sprache(n) aus. Zum Übersetzen wählen Sie in einer Box die Sprache der Original-Hilfe, in der anderen Box die Sprache, in die die Hilfe übersetzt werden soll. Wird eine der Sprachen nicht angezeigt, dann schliessen Sie bitte diesen Dialog und geben zuerst den Sprachnamen im Dialog Sprache hinzufügen ein.

Überschrift

Geben Sie das Hilfethema, respektive die Übersetzung ein.

Hilfetext

Geben sie den Hilfetext - respektive seine Übersetzung - in das MLE ein.

Blättern

Mit dem <<- bzw. >>-Knopf ganz unten im Dialog können Sie durch die Hilfetexte blättern. Die Nummer des angezeigten Hilfetextes kann im Eingabefeld zwischen beiden Knöpfen abgelesen werden. Wird kein Hilfetext angezeigt, dann existiert für die gewählte Hilfenummer noch kein Text oder diese Nummer wird im Programm nicht verwendet.

Kopieren

Mit dem Druck auf kopieren wird der Inhalt des linken MLEs in das rechte MLE kopiert. Das kann in manchen Fällen recht nützlich sein. (Die Grösse des Knopfes wurde inzwischen der Länge des deutschen Wortes angepasst...)

Sichern

Mit dem Druck auf Schliessen wird der Inhalt des Feldes automatisch gesichert.

Topics

Den Dialog zur Eingabe und Übersetzung der Topics erreichen Sie über Tools - OnLine-Hilfe. Alle Eingaben werden automatisch in das Topics-Feld geschrieben, sobald das gerade bearbeitete Dialogelement den Eingabefokus verliert.
Die Bedienung des Topics-Dialogs entspricht dem oben beschriebenen OLH-Dialog.
