Standardfelder

Dieses Menü ermöglicht das Anlegen aller Standardfelder für ein neues Programm in einem Rutsch. Drei weitere Menüpunkte öffnen Dialoge zur komfortablen Eingabe der Daten in Drehknopf-, Eingabefelder-, Bitmap- und Notizbuch-Kontrollfeldern.

Alle Erzeugen

Die Auswahl des ersten Menüpunktes öffnet folgenden Dialog:
Dieser Dialog wird immer mit den zuletzt eingegebenen Werten geöffnet, um die Tipparbeit auf das absolut notwendige Minimum zu reduzieren. Haken Sie die benötigten Felder in der Liste ab und bestimmen Sie die benötigten Einträge der einzelnen Felder. Die Nummer des Notizbuchfeldes wird standardgemäss auf FFFFFF00 gesetzt. Sollte Ihr Programm mehrere Notizbücher enthalten, ist die Nummer des anzulegenden Notizbuchs im Eingabefeld entsprechend abzuändern.

FFFFFF12

Enthält die Texte für Menüs und Dialoge. Kleinere Applikationen können mit 256 Einträgen auskommen, für mittlere Applikationen (zum Beispiel DatTools) sollten mindestens 512 Einträge gewählt werden.

FFFFFF13

Das Feld wird direkt aus dem DatTools-Ordner kopiert, da die dort gespeicherte Version immer dem aktuellsten Stand entspricht.

nsm.dat

Enthält die Nachrichten der Anwendung. Die Einträge 0x00 (ERR_INVALID) und 0x01 (ERR_NO_ERROR) werden automatisch auf die vorgegebenen Standardtexte gesetzt.

err.dat

Das Feld wird direkt aus dem DatTools-Ordner kopiert, da die dort gespeicherte Version immer dem aktuellsten Stand entspricht.

olh.dat

Enthält die Texte der Onlinehilfe. Die Hilfetexte der Bibliotheksfunktionen werden dabei automatisch in das neue Feld kopiert. Die Anzahl der Einträge sollte so grosszügig bemessen werden, dass für spätere Erweiterungen Ihrer Anwendung noch ausreichend Platz zur Verfügung steht.

top.dat

Enthält die Texte der Topics. Die in Hilfetexten der Bibliotheksfunktionen verwendeten Topics werden automatisch in das neue Feld kopiert. Die Anzahl der Einträge richtet sich ausschliesslich nach Ihrem persönlichen Bedarf.

FFFFFF01

Dieses Feld wird nur dann benötigt, wenn Sie ST-Open's erweiterte Drehknöpfe in Ihrem Programm verwenden. Wählen Sie die benötigte Anzahl an Drehknöpfen. Es ist ratsam, eine Reserve für spätere Erweiterungen einzuplanen.

FFFFFF10

Dieses Feld wird nur dann benötigt, wenn Sie ST-Open's Bitmapfunktionen verwenden. Wählen Sie die benötigte Anzahl an Bitmaps. Es ist ratsam, eine Reserve für spätere Erweiterungen einzuplanen.

FFFFFF00

Dieses Feld wird nur dann benötigt, wenn Sie ST-Open's Notizbuchfunktionen verwenden. Wählen Sie die für Ihr Notizbuch benötigte Anzahl von Seiten.
