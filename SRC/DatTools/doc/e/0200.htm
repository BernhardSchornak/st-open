Drives

To work with DatTools, you need a source folder where fields are read from, written to or put in. The first time you start DatTools, you should select the drives DatTools scans for valid source folders. To activate these drives, choose Select in the main menu, then Drives:
The drive selection dialog is opened:
Activate the drive where your V700 distribution is stored. In general, this is just one drive. Activating specific drives speeds up scanning for valid source folders markably. Please do not activate CD-ROM or DVD drives - it takes a lot of time to scan them and produces a lot of very annoying messages if no media is present.

Source Folder

If you choose Source Folder from the menu, DatTools scans all activated drives for "ST7" folders. Old distributions (ST4, ST5, ST6) are ignored because data fields of versions prior to version 7.0.0. are incompatible with the new field header definition. Found source folders are shown in the selection dialog:
Select a source folder of your choice - either double-click on the name or select the listbox item and push Select. This selected folder is the new working directory. Fields are loaded from here, altered fields are written back to and new fields are put into the DATA or SYST subfolders residing in this folder.

Fields

Choose the menu item Field to display a list with all fields found in the source folder:
Double-click on any name or select one, then push Select to make the selected field the working field. This selection is required if you want to view and edit a field's content, move or copy entries or open the spinbutton, bitmap or notebook field editor.
