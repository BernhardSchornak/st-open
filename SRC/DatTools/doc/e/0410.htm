OnLine-Help (OLH)

All MLEs have the size of the original OLH or Topics. This design visualises how entered text will look like in the opened help dialog. To enter help texts or topics in only one language, it's more comfortable to write them into the application's help dialogs - just add the line PGM_MODE = MODE_CODING before you call LDinit().

Help Texts

You can open the dialog for editing and translating help texts via Translation - OnLine-Help. The entered text is copied to the OLH field whenever the currently edited dialog element loses the input focus.

Language Selection

Choose the desired language(s) with the << or >> button(s) left of the entryfield(s). For translations select one box with the language the original help text is written in and the other box with the target language. If your language isn't amongst the listed languages: Close this dialog and add your language in the languages dialog, then open this dialog again.

OLH Title

Type the heading for the help or your translation into one of the entryfields.

OLH Text

Type the help text or your translation into one of the MLEs.

Help Numbers

Use the << or >> button at the bottom of the dialog to switch between the available help numbers. The number of the currently displayed help is shown in the entryfield between both buttons. If no help text is displayed, there's either no text typed in yet or the help number is not defined in the program.

Copy

Push the copy button to copy the content of the left MLE to the right MLE. Sometimes this might be a quite useful feature.

Save

If you push the Close button, the content of the OLH field is saved automatically.

Topics

To open the dialog for editing and translating OLH Topics select Translation - OLH Topics. Entered text is copied to the Topics field whenever the corresponding dialog element loses the input focus.
