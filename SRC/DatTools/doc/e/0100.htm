DatTools

- are a collection of tools to create and edit data fields used by ST-Open's database engine (DBE).
- provide easy to use translation tools for multi-lingual programs. ST-Open's libraries support up to 16 languages (menus, dialogs, messages and online help). Users can switch from one language to another with a simple click in the running program.
- are written in 100% hand-optimised assembler code - a guarantee for much higher code density and outstanding performance.
- are 100% compliant to the rules of Intelligent Design. With 21st century's programming techniques, programs become smaller and faster than ever.
Note 1: DatTools make heavy use of XMM registers. A processor with SSE or 3Dnow! extension is required to run this program.
Note 2: Intelligent Design is an optimisation for AMD's Athlon processor family. You cannot experience the maximum speed boost of Intelligent Design code if you run it on less sophisticated processors with only one or two integer execution pipes per CPU.

Language

DatTools are multi-lingual. Each language field can hold up to 16 languages. You can switch to any of the available languages in the running program as often as you want. To switch to another language select Program, then Language to open a list with all available languages:
Select the language of your choice. All menus, dialogs, messages and help texts are changed to the new language immediately. Open dialogs are not concerned - you have to close them manually, then open them again to work in the new language.

Encryption

To edit encrypted data fields, you have to activate ST-Open's Security System (SecSys), first. Select Program, then SecSys Login to start SecSys. You need the master key as well as the PIN and password of a user with unrestricted write permission to edit encrypted fields.
