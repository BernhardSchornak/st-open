Tools

DatTools' translation tools are accessible via Translation in the main menu:
  • Languages
  • OnLine-Help (OLH)
  • Menus and Dialogs
  • Program Messages
  • System Dialogs
  • System Errors

Languages

To add another language or edit an existing language name, select Languages in the Translation menu:
The dialog shows the content of entries 0x20 through 0x2F (stored in the field STR.DAT). Empty entryfields signal there's no language defined and no translation exists yet. Enter the name of the new language in the next free entryfield and push Close to save the changes and dismiss the dialog.
If all entryfields are filled with language names, there are two possible options. Either you may
1. replace one of the existing languages
or
2. you may create a new set of fields (FFFFFF12 (Menus), FFFFFFF5 (Messages), FFFFFFFB (OLH), FFFFFFFA (Topics)).
The second method is more flexible and consistent (an automated function to perform this task is on my todo list).
