Disk Image

- Kopiert den Inhalt einer Abbild-Datei (*.img) auf einen Datenträger.
- Zeigt den Inhalt beliebiger Sektoren auf beliebigen Laufwerken an.
- Besteht zu 100 Prozent aus handoptimiertem Assemblercode.
Hinweis 1: Dieses 64 Bit Programm verwendet SSE2-Befehle. Ein entsprechender 64 Bit Prozessor und eine 64 Bit Version von Windows sind zur Ausführung zwingend erforderlich.

Sprache

Disk Image ist - wie alle ST-Open-Programme - mehrsprachig. Wie bei allen ST-Open-Programmen ist die Sprachauswahl das erste Untermenü im Menü Programm. Klappen Sie das Untermenü Sprache (im Auslieferungszustand steht dort Language) auf, um die im Programm verfügbaren Sprachen anzuzeigen:
Wählen Sie eine der angezeigten Sprachen. Alle Menüs, Dialoge, Nachrichten und Hilfetexte werden verzögerungsfrei auf die gewählte Sprache umgeschaltet. Offene Dialoge sind von der Sprachauswahl nicht betroffen. Sie müssen manuell geschlossen und wieder geöffnet werden, um in der neuen Sprache arbeiten zu können.
