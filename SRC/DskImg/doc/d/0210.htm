Quell-Auswahl

Um ein Abbild auf einen Datenträger schreiben zu können, muss dem Programm mitgeteilt werden, wo das zu schreibende Abbild abgelegt ist. Wählen Sie im Menü Abbild den Menüpunkt Quell-Auswahl:
Der übliche Datei Öffnen-Dialog wird angezeigt. Wählen Sie das Laufwerk, den Pfad und die Datei, die das Abbild enthält:
Ein Doppelklick oder das Betätigen des Knopfes Öffnen wählt die gewünschte Datei aus. Ein Klick auf Abbrechen schliesst den Dialog ohne Auswahl, und setzt das Quelle gewählt-Kontrollfähnchen zurück.
Anmerkung: Ohne Auswahl einer Quelle kann kein Abbild geschrieben werden.
