Abbild Schreiben

Nach der Auswahl von Quelle und Ziel wählen Sie Abbild Schreiben im Untermenü Abbild
um den Abbild Schreiben Dialog anzuzeigen. Sofern Quelle oder Ziel noch nicht ausgewählt wurden, wird statt dessen jedoch eine Fehlermeldung erscheinen.
Wenn Sie Sich nicht wirklich sicher sind, was Sie gerade tun: Drücken Sie Abbrechen, um den Dialog zu schliessen!
Wenn Sie sicher sind, dass Sie das gewünschte Abbild gewählt haben, und das Ziel-Laufwerk keine wichtigen Daten enthält, ist es an der Zeit, auf Schreiben zu drücken.
Wie zuvor erwähnt, legt DskImg zuerst eine Sicherheitskopie der überschriebenen Sektoren an. Beachten Sie bitte, dass auf dem Laufwerk, auf dem Sie DskImg abgelegt haben ausreichend Platz für diese Sicherheitskopie ist. Schlägt das Sichern fehl, wird kein Abbild auf das Ziel-Laufwerk geschrieben! Sollte das der Fall sein, verschieben Sie den gesamten DskImg Ordner bitte auf ein Laufwerk mit ausreichender Kapazität. Ausreichend heisst: Es muss mindestes soviel Platz frei sein, dass eine Datei mit der Grösse der Abbild-Datei auf das Laufwerk passt!
Nach erfolgter Sicherung beginnt DskImg mit dem Schreiben des Abbilds. Schlägt dieser Versuch fehl, versucht DskImg, das Laufwerk zu demontieren und zu sperren, bevor es einen zweiten Versuch unternimmt, das Abbild zu schreiben. Schlägt der zweite Versuch ebenfalls fehl, besteht wahrscheinlich keine Möglichkeit, Daten auf das Laufwerk zu schreiben. Wenn Sie es mit Gewalt versuchen wollen: Löschen Sie sämtliche Daten und Partitionen auf dem Laufwerk und versuchen Sie es erneut. Sollte es dann immer noch nicht klappen - versuchen Sie es einmal mit einer anderen Software...
Hat DskImg keine Fehler gemeldet, und "Fertig!" wird angezeigt, sollten Sie nun eine startbare Kopie des Abbilds auf Ihrem Laufwerk haben. Fahren Sie Windows herunter, Öffnen Sie das BIOS Menü beim hochfahren, machen Sie das erstellte Laufwerk zum Startlaufwerk und starten Sie den Computer erneut...
