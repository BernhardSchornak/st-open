Write Image

After selecting source and target, pick Write Image from the Image submenu
to pop up the Write Image dialog. If source or target were not selected, yet, an error message pops up, instead.
If you are not shure what you are doing here: Push Abort to dismiss this dialog!
If you are sure you selected the source of your choice and your target drive only holds unimportant data which can be overwritten without a risk, it is time to push the Write button.
As mentioned before, DskImg creates a safety copy of the overwritten sectors, first. Keep care: The drive where DskImg resides must have sufficient capacity to store the backup. If saving fails, no image is written! If so, move the DskImg folder to a drive with sufficient capacity. Sufficient means: At least the size occupied by the image file must be free!
When saving succeeded, DskImg starts to write the image. If the first attempt fails, it tries to lock and unmount the drive, then retries to write. If that fails, there is no chance to write the image to that drive. If you want to force to use it: Delete its entire content and all partitions, then retry. If it still does not work - try another image writing software...
If DskImg did not report any errors and 'Done!' is shown, your target drive should hold a bootable copy of the image, now. Shut down Windows, open BIOS settings while rebooting, select the drive as boot drive and reboot...
