Source Selection

To write an image to a drive, the program should know where that image can be found. Click on Image in the main menu and select the option Image Selection:
The standard File Open dialog pops up. Choose the drive and path to the image file and select that file:
Doubleclick the file or push Open to mark this file as source. Push Abort to reset the source present flag and dismiss the dialog.
Note: You cannot write an image if no source is selected.
