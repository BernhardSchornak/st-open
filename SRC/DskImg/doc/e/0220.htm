Select Target

The second step is the selection of a target drive. Pick Target Selection from the Image submenu
to open the Target Selection dialog:
You can see a list with up to ten physical drives currently attached to your PC. DskImg automatically retrieves these data when it is started (stored in data\00000001). The columns show the name, media type, cylinders (tracks for floppies), heads, sectors, sector size and capacity of each drive. Media type can be one of these:
  • 5.25/1.2M various floppy disk formats
  • [x] fixed drives
  • |-> removable drives
  • ??? unknown media
When the dialog pops up, the topmost radio button is selected per default, but the 'selected' drive is set to invalid until you really select a drive (just push one of those radio buttons). If you push Abort, the selected drive is set to invalid and the target present flag is reset. Push Select to choose the selected target drive and set the target present flag.
Note: You cannot write an image if no target is selected.

Advice

Do yourself a favour and leave your hands off of fixed disks. It is no good idea to select a non-removable harddisk as target. If the HD has more than one partition, it is very likely that DskImg cannot write to the drive, because it only unmounts and locks the physical drive, while logical volumes residing on that drive still are mounted and unlocked. Moreover, you risk to lose all data stored on the drive if DskImg succeeds and writes the image. Consider to 'sacrifice' a thumb drive with no important data on it...
