Disk Image

- Writes a selected image file (*.img) to a selected drive.
- Provides a simple sector viewer.
- Is written in 100% hand-optimised assembler code.
Note 1: Disk Image makes heavy use of XMM registers. A processor with SSE2 extension is required to run this program.

Language

Disk Image is multi-lingual. As standard for all ST-Open programs, the language selection submenu is the first submenu of Program. Select Program - Language to open a list with all available languages:
Select the language of your choice. All menus, dialogs, messages and help texts are changed to the new language immediately. Open dialogs are not concerned - you have to close them manually, then open them again to work with the new language.
