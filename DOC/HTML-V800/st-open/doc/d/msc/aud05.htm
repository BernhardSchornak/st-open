Ausgabe
st2mo() wandelt die stereophonen Daten der [quelle] in monaurale Daten um und speichert diese im [ziel]. Das [ziel] kann halb so gross wie die [quelle] sein, da die Datenmenge halbiert wird. Die Lautstärke für den Ausgangskanal kann eingestellt werden. Es werden sowohl 8- als auch 16-Bit Dateien verarbeitet.

[quelle] muss einen gültigen Wellenkopf enthalten, da dieser für die Verarbeitung benötigt wird. [ziel] enthält nach der Umwandlung eine gültige Wellendatei.
Beschreibung
DD MeineFunktion(VOID *quelle, VOID *ziel)
{
 DD rc;
 rc = st2mo(quelle, ziel, 0x40000, 100);
 return(rc);
}
256 kB aus [quelle] werden in eine 128 kB grosse Monowellendatei umgewandelt und in [ziel] gespeichert. Die Lautstärke ist 100%.
