Ausgabe
m2s() wandelt die monaurale Quelldatei in eine stereophone Zieldatei um. Die Lautstärke des linken und rechten Kanals kann zwischen 0 und 100 Prozent gewählt werden.
Beschreibung
DD MeineFunktion(VOID)
{
 DD rc = m2s("mono.wav", "stereo.wav", 100, 100);
 return(rc);
}
Wandelt die monaurale Datei "mono.wav" in die stereophone Datei "stereo.wav" um. Die Lautstärke beider Kanäle ist 100 Prozent.
