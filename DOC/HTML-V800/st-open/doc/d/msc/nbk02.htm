Ausgabe
SetTab() berechnet die Grösse der Reiter und setzt die entsprechenden Fensterparameter.
Beschreibung
DD MeineFunktion(HWND hwnd)
{
 DD rc;
 rc = SetTab(hwnd);
 return(rc);
}
Dies ist eine interne Funktion, die von InitBk() automatisch aufgerufen wird.
