Ausgabe
mo2st() wandelt die monauralen Daten der [quelle] in stereophone Daten um und speichert diese im [ziel]. Das [ziel] muss mindestens doppelt so gross wie die [quelle] sein, da die Datenmenge sich verdoppelt. Die Lautstärke für die beiden Ausgangskanäle kann getrennt eingestellt werden. Es werden sowohl 8- als auch 16-Bit Dateien verarbeitet.

[quelle] muss einen gültigen Wellenkopf enthalten, da dieser für die Verarbeitung benötigt wird. [ziel] enthält nach der Umwandlung eine gültige Wellendatei.
Beschreibung
DD MyFunction(VOID *quelle, VOID *ziel)
{
 DD rc;
 rc = mo2st(quelle, ziel, 0x20000, 100, 100);
 return(rc);
}
128 kB aus [quelle] werden in eine 256 kB grosse Stereodatei umgewandelt und in [ziel] gespeichert. Die Lautstärke beider Stereokanäle ist 100%.
