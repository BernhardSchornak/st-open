Ausgabe
s2nor() wandelt die übergebene Zeichenkette in das normalisierte interne Rechnerformat um und legt das Ergebnis im angegebenen Puffer ab.
Beschreibung
DD MeineFunktion(VOID *adr)
{
 DD rc;
 rc = s2nor(adr, 0);
 return(rc);
}
OP1 enthält nun die aus der Zeichenkette extrahierte und normalisierte Zahl.
