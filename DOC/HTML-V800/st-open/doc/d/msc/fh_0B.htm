Funktion
rc = DsrchP(muster, SpeiHenk, UntFeld, start, anzahl);

Eingabe

00000000 Fehler oder keine Übereinstimmung

Ausgabe
DsrchP() durchsucht das Dateisystem nach dem übergebenen Muster. Bis zu [anzahl] gefundene Verzeichnisse (inklusive Pfad) werden im angegebenen Unterfeld des angegebenen Feldes gespeichert.
ACHTUNG: Ist [start] Null, so wird das gesamte Feld wird vor dem Suchen gelöscht!
Beschreibung
DD MeineFunktion(DD mh) 
{
 DD rc;
 rc = DsrchP("meins", mh, 0, 0, 256);
 return(rc);
}
Das aktuelle Verzeichnis wird nach Verzeichnissen "meins" durchsucht. Da [start] Null ist, wird das Feld vor der Suche gelöscht. Bis zu 256 gefundene Verzeichnisnamen (mit Pfad) werden im Feld [mh], Unterfeld 0 gespeichert. Der Rückgabewert rc enthält die Anzahl gefundener Verzeichnisse. Ist er Null, dann wurde kein Verzeichnis gefunden, bzw. ein Fehler führte zum Abbruch der Funktion.
