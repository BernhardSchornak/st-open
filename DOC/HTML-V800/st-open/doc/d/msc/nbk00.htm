Ausgabe
InitBk() initialisiert das Notizbuch mit den im Notizbuchfeld enthaltenen Daten und zeigt es dann auf dem Bildschirm an. Alle relevanten Daten werden von dieser Funktion verwaltet.
Beschreibung
DD MeineFunktion(HWND hwnd, DD mh)
{
 DD rc;
 rc = InitBk(hwnd, mh);
 return(rc);
}
Das Notizbuch sollte nun auf dem Bildschirm erscheinen.
