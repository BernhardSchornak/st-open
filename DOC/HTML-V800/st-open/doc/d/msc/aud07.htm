Ausgabe
c16x8() wandelt 16 Bit Proben der [quelle] in 8 Bit Proben um und speichert diese im [ziel]. Das [ziel] kann halb so gross wie die [quelle] sein, da die Datenmenge sich halbiert. Die Funktion kann sowohl Mono- als auch Stereo-Dateien verarbeiten.

Die übergebenen Adressen [quelle] und [ziel] müssen auf den Anfang des Datenbereichs verweisen, es können somit beliebige Teile innerhalb der Datei umgewandelt werden.
Beschreibung
DD MyFunction(VOID *quelle, VOID *ziel)
{
 DD rc;
 rc = c16x8(quelle, ziel, 0x2000);
 return(rc);
}
8.192 16 Bit Proben aus [quelle] werden in 8 Bit Proben umgewandelt und in [ziel] gespeichert.
