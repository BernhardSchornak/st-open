Ausgabe
mix16() mischt bis zu 32 Quellwellendateien zu einer Zielwellendatei. Die Anzahl der Quelldateien muss 2, 4, 8, 16 oder 32 sein, andere Mischoperationen werden nicht unterstützt. Das Format aller Quelldateien muss identisch sein, gemischte Formate werden zurückgewiesen.
Beschreibung
DD MeineFunktion(VOID *array, VOID *ziel)
{
 DD rc;
 rc = mix16(array, ziel, 8, 0x2000);
 return(rc);
}
Mischt die 8 in [array] spezifizierten Wellendateien zu einer 8.192 Byte grossen Wellendatei [ziel].
