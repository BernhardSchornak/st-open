FLSdlg() ist die Dialogprozedur der Dateiliste. Der Dialog selbst muss mit DBox() oder dbox() geöffnet werden. Sie sollten dbox() nur in Ausnahmefällen verwenden, da OS/2 die Resourcen von Dialogen mit dem Elternhenkel des Schreibtisches nicht mehr freigibt - bei jedem Aufruf "verschwindet" ein Teil des verfügbaren Speichers...
Ist die Systemvariable STV_FLS_ADD ungleich Null, so wird der Dateiname an die Spitze der Dateiliste geschrieben. Alle anderen Einträge der Dateiliste werden hierbei nach unten verschoben.
Beschreibung
DD MeineFunktion(HWND h) 
{
 ...
 DBox(h, (PFNWP)FLSdlg, ID_FLST);
 if(FDacc(FLD_STR, STR_FLS_SELECT, 0, FDA_OFFSET, 0, 0))
 {
  /* Datei-Auswahl verarbeiten...  */
 }
 else
 {
  /* "ABBRUCH" wurde gedrückt */
 }
 ...
}
DBox() öffnet den Dialog FLSdlg() mit der Resource-ID ID_FLST. Der Dialog verwaltet alle möglichen Aktionen der Dateiliste. Wird eine Datei mittels Öffnen oder per Doppelklick ausgewählt, dann wird der selektierte Dateiname in die Systemzeichenkette STR_FLS_SELECT kopiert. Wird Abbrechen gewählt, dann wird eine leere Zeichenkette zurückgeliefert. Der Dialog wird in beiden Fällen automatisch geschlossen.
