Eingabe
DD
rc
00000000 kein Fehler
0000003A ERR_NAME_OBJECT
Ausgabe
NameObj() überschreibt den Objekttitel der Datei mit dem übergebenen Namen.
Beschreibung
DD MeineFunktion(VOID) 
{
 DD rc;
 rc = NameObj("C:\config.sys", "CONFIG.BAK");
 return(rc);
}
Die CONFIG.SYS wurde in CONFIG.BAK umbenannt, sofern rc Null ist. Beachten Sie, dass beim Dateinamen Gross- und Kleinschreibung von OS/2 ignoriert werden (cOnFiG.sYs würde zum selben Ergebnis führen). Der neue Name wird im Gegensatz dazu so übernommen, wie er übergeben wurde.
