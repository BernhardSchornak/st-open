Die Datenbankmaschine

ST-Open's Datenbankmaschine ist mit Abstand die grösste Funktion im ST-Open System. Sie kann bis zu 4.294.967.295 Felder mit einer maximalen Grösse von jeweils 4.294.967.295 Byte verwalten. In der Praxis ist die Grösse aller geladenen Felder allerdings auf den allozierbaren Speicher beschränkt, so dass die im wahren Leben verwendeten Felder wohl deutlich unterhalb des Limits liegen. Dieses Tutorial vermittelt Ihnen einen Überblick über den inneren Aufbau der Datenbankmaschine und des Feldkopfes. Weiterhin wird durch Codebeispiele die Anwendung der einzelnen Funktionen anschaulich demonstriert.
