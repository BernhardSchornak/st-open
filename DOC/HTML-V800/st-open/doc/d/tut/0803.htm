SecSys starten

Um das Sicherheitssystem zu starten, muss zuerst der Befehl SS_PRP ausgeführt werden: 
SecSy(SS_PRP, 0, 0);
Dieser Befehl alloziert den SecSys-Arbeitsbereich und setzt alle Variablen auf festgelegte Startwerte. Beim zweiten Aufruf muss der Befehl SS_CKM übergeben werden: 
SecSy(SS_CKM, 0, 0);
Mit diesem Befehl wird das Meisterschlüsselfeld geladen und der Anmeldedialog für den Systemmeister wird aufgerufen. Nach erfolgter Anmeldung wird der Meisterschlüssel aus dem Meisterschlüsselfeld zusammengesetzt und in den Arbeitsbereich kopiert. Das Sicherheitssystem ist nun für den letzten Schritt bereit, der den Befehl SS_RUN an das Sicherheitssystem sendet: 
SecSy(SS_RUN, 0, 0);
Das Sicherheitssystem läuft nun und bewacht Ihre Daten.

Erstanmeldung

Wenn Sie ein durch SecSys geschütztes Programm das allererste mal starten, fragt das System nach einem Meisterschlüssel. Geben Sie ein Passwort Ihrer Wahl sowohl in das obere als auch in das untere Eingabefeld ein. Beide Eingaben müssen identisch sein! Nach erfolgreicher Anmeldung wird der Anmeldedialog für Benutzer eingeblendet. Bitte geben Sie die PIN (persönliche Identifikationsnummer) und das Passwort ein, mit dem Sie Sich zukünftig im System einloggen wollen. Wenn Sie Sich Passwörter und Nummern schwer merken können, schreiben Sie die Anmeldedaten auf und bewahren Sie das Schriftstück an sicherer Stelle auf - ohne gültige Anmeldedaten können Sie weder das Programm öffnen, noch auf Ihre geschützten Datenfelder zugreifen!

Technische Anmerkung

Der Startvorgang ist in drei Schritte aufgeteilt, um Anwendungsprogrammierern Gelegenheit zu geben, diverse Vorbereitungsschritte einzufügen, die zwischen den drei Aufrufen erledigt werden müssen. Zudem ist diese Aufteilung für die geplante Erweiterung auf verteilte Arbeitsplätze erforderlich, da an diesen Stellen die Kommunikation der miteinander verbundenen Rechner eingefügt werden soll.
