FDA_COMPARE

vergleicht den gegebenen Eintrag mit dem übergebenen Muster. Die Funktion liefert Null zurück, wenn Eintrag und Muster identisch sind, eine 1, wenn der Eintrag grösser als das Muster und eine 2, wenn das Muster grösser als der Eintrag ist.

Numerische Daten

Bytes, Worte und Doppelworte

DD Meine_Funktion(DD mh)
{
 DD a;
 a = FDacc(mh, et, sub, FDA_COMPARE, 0, 0x12345678);
 return(a);
}

Vier- und Achtfachworte

DD Meine_Funktion(DD mh)
{
 DQ q;
 DD a;
 q = 0x0123456789ABCDEF;
 a = FDacc(mh, et, sub, FDA_COMPARE, &q, 0);
 return(a);
}

Zeichenketten

DD Meine_Funktion(DD mh)
{
 DD a;
 a = FDacc(mh, et, sub, FDA_COMPARE, "Vergleich mich!", 0);
 return(a);
}
