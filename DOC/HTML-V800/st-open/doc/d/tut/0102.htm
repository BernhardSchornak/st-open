LD

Die LD-Struktur wird zur Speicheranforderung mit ST-Open's Lader benötigt. Nach dem Aufruf von LDreq() kann sie bedenkenlos gelöscht werden, da alle weiteren Zugriffe über den Speicherhenkel (LD.MMoff) erfolgen. Der Speicherhenkel und gegebenenfalls die Adresse des Speicherblocks (LD.MemEA) sollten in globalen Variablen abgelegt werden, wenn der Speicherblock funktionsübergreifend verwendet werden soll. Der von LDreq() zurückgelieferte Speicherhenkel muss auf jeden Fall gespeichert werden, da er für alle Laderfunktionen benötigt wird. Sie können den Speicherblock ohne Speicherhenkel nicht einmal freigeben, da er der einzige Aufrufparameter für LDfre() ist.
Beschreibung
   typedef struct _LD{
                      VOID *MemEA;
                      DQ    MemSz;
                      DD    FldNr;
                      DD    LdCtl;
                      DQ    MMoff;
                      VOID *FileN;
                      DQ    Moffs;
                      DQ    Foffs;
                      DQ    AmtBy;
                      VOID *FileH;
                     } LD;
   typedef LD *PLD;
