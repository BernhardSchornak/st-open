ST-Open's Drehknöpfe

STspn() stellt neben dem von OS/2 bereitgestellten Standarddreknopf neun weitere Drehknöpfe zur Verfügung:
32 Bit als formatierte Dezimalzahl, wahlweise im Pseudofliesskommaformat
64 Bit als formatierte Dezimalzahl, wahlweise im Pseudofliesskommaformat
08 Bit als formatierte Hexadezimalzahl
16 Bit als formatierte Hexadezimalzahl
32 Bit als formatierte Hexadezimalzahl
64 Bit als formatierte Hexadezimalzahl
Zeit (formatiert)
Datum (formatiert)
Zeichenketten aus einem Feld
Zeichenketten aus einem übergebenen Zeichenkettenarray
