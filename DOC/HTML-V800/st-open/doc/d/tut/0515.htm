FDA_DEC

dekrementiert den gegebenen Eintrag. Eine Bereichsüberprüfung wird vor dem Dekrementieren nicht vorgenommen - es ist Sache des Programmierers, alle Ergebnisse auf Gültigkeit zu überprüfen. Das Ergebnis der Operation wird bei den Typen 01 - 03 zurückgegeben. Bei den Typen 04 und 05 ist der Rückgabewert prinzipiell Null, da das Ergebnis nicht in ein 32 Bit Register passt.
FDA_DEC ist nur für die numerischen Datentypen 01 bis 05 verfügbar. Bei allen anderen Datentypen überspringt die DBE die Auführung und kehrt direkt zum Aufrufer zurück.

Numerische Daten

Bytes, Worte und Doppelworte

DD Meine_Funktion(DD mh)
{
 DD a;
 a = FDacc(mh, et, sub, FDA_DEC, 0, 0);
 return(a);
}

Vier- und Achtfachworte

DQ Meine_Funktion(DD mh)
{
 DQ q;
 FDacc(mh, et, sub, FDA_DEC, &q, 0);
 return(q);
}

Intelligent Design

Parameter 5 ist nur für Adressen erforderlich. Parameter 6 wird nicht benötigt.
