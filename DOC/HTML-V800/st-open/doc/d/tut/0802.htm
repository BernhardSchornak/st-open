SecSy()

SecSys wird mit einem einzigen Funktionsaufruf gesteuert: 
SecSy(Befehl, Adresse, Wert);
Einer der unten aufgeführten Befehle.
Befehl
optional: Adresse einer Zeichenkette.
Adresse
optional: 32 Bit breiter numerischer Wert.
Wert

Befehle

Befehle in Klammmern sind nicht implementiert oder setzen mit Null initialisierte Variablen auf Null zurück. Alle nicht definierten Befehle werden intern auf den Wert SS_ERR gesetzt, bevor der Funktionsverteiler angesprungen wird.
