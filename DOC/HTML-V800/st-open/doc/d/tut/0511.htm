FDA_ADDRESS

ermittelt die Adresse eines Eintrags. Der Aufruf ist für alle Datentypen identisch. 
VOID * Meine_Funktion(DD mh)
{
 VOID *p;
 p = FDvpt(mh, et, sub, FDA_ADDRESS, 0, 0);
}
Bitte beachten Sie, dass der Funktionsname für die Rückgabe von Adressen in FDvpt() umbenannt wurde. Sie rufen auch weiterhin FDacc() auf: 
       .align 2,0x90
.globl _FDvpt
_FDvpt:
.globl _FDacc
_FDacc:subl $0x40,%esp
Die Definition des Rückgabewertes ist von "Lang" jedoch in "Zeiger auf Leer" umbenannt, um dem C-Kompilierer den Umgang mit 32 Bit breiten Daten identischen Inhalts etwas zu erleichtern.

Intelligent Design

Parameter 5 und 6 sind nicht erforderlich. In reinen Assemblerprogrammen können wir durchgehend _FDacc verwenden, ohne auf die C-Krücke FDvpt() zurückgreifen zu müssen.
