Ausgabe
tSum() addiert die übergebenen Zeiten und liefert das Ergebnis zurück. Das Ergebnis ist keine gültige Zeitangabe! Übersteigt 255 Stunden, dann wird die Stunde auf Null gesetzt.
Beschreibung
DD MeineFunktion(VOID)
{
 DD zeit1 = 0x173B3B63, /* 23:59:59.99 */
    zeit2 = 0x173B3B63, /* 23:59:59.99 */
    rc;
 rc = tadd2(zeit1, zeit2);
 return(rc);
}
rc enthält nun 0x2F3B3B62, das sind 47 Stunden, 59 Minuten, 59 Sekunden und 98 Hundertstelsekunden.
