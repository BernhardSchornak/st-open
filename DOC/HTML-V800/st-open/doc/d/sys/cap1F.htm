Ausgabe
WInvR() ruft WinInvalidateRect() auf und stellt anschliessend den vom API zerstörten Inhalt der Register ECX und EDX wieder her.
Rufen Sie diesen Sandkasten nicht von C aus auf. Schlampige Programmierumgebungen profitieren nicht von wiederhergestellten Registern. Sie verschwenden etliche Taktzyklen für das unnötige Wiederherstellen von Registerinhalten, die sofort wieder mit aus dem Speicher geladenen Parametern überschrieben werden.
Beschreibung
{
 WInvR(hwnd, 0, 0);
}
Das Rechteck des Fensters hwnd wird in die Liste ungültiger Bildschirmbereiche aufgenommen und eine WM_PAINT-Nachricht wird an die Fensterprozedur gesendet.
