Ausgabe
W2hex() wandelt das untersten 16 Bit des übergebenen Doppelwortes in eine ASCII-Zeichenkette um. Die Bits 63 bis 16 werden ignoriert!
Der übergebene Puffer muss eine Midestgrösse von acht Byte haben!
Beschreibung
{
 DB s[16];
 W2hex(0x12345678, s);
}
Die Zeichenkette s enthält nun "5678", die oberen 48 Bit des Vierfachworts werden ignoriert.
