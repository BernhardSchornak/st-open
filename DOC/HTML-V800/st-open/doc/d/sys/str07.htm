SIns
Funktion
SIns(ziel, quelle, start);
Aufruf
VOID *
ziel
Adresse der Zielzeichenkette
VOID *
quelle
Adresse der Quellzeichenkette
DD
start
Index an dem die Quellzeichenkette eingefügt werden soll
Eingabe
DD
rc
Immer Null
Ausgabe
SIns() fügt die Quellzeichenkette an Index [start] der Zielzeichenkette ein. Achten Sie darauf, dass die Zielzeichenkette ausreichend gross dimensioniert ist!
Beschreibung
VOID MeineFunktion(VOID)
{
 DD rc;
 DB s[32];
 SCpy(s, "Verteilerdose");
 SIns(s, "lose Verteiler", 9);
 return(rc);
}
s enthält nun die Zeichenkette "Verteilerlose Verteilerdose"
