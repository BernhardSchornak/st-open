hex2Q
Funktion
rc = hex2Q(quelle);
Aufruf
VOID *
quelle
Adresse der Quellzeichenkette
Eingabe
DD
rc
immer Null
Ausgabe
hex2Q() wandelt die Zeichenkette an der Adresse [quelle] in ein Vierfachwort (8 Byte) um und gibt das Ergebnis im Quellpuffer zurück. Die Quelle wird rückwärts ausgewertet, überzählige Stellen werden ignoriert. Ein führendes Minuszeichen wird als Vorzeichen erkannt, die Zahl wird ins Zweierkomplement umgesetzt. Beachten Sie, dass das Zweierkomplement von Zahlen grösser 0x7FFFFFFFFFFFFFFF eine positive Zahl ist!
Beschreibung
{
 DD rc;
 DB s[16];
 SCpy(s, "10 Gabeln");
 hex2Q(s);
}
Die Zeichenkette s enthält nun die Bytes 0xBE 0x0A 0x01 0x00 0x00 0x00 0x00 0x00 (ein gültiges Vierfachwort, das z.B. in Rechenoperationen verwendet werden kann).
