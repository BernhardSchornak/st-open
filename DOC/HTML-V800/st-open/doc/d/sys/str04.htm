Aufruf

Anzahl zu entfernender Bytes
Eingabe
DD
rc
Immer Null
Ausgabe
SCut() entfernt [anzahl] Bytes ab Index [start] aus der übergebenen Zeichenkette. Der gültige Bereich für den Index ist 0 bis [Länge - 1]. Liegt [start] + [anzahl] jenseits der abschliesenden Null, dann werden nur die Zeichen bis zur abschliessenden Null entfernt.
Beschreibung
VOID MeineFunktion(DB *s)
{
 SCut(s, 0, 3);
}
Schneidet die ersten drei Bytes aus der Zeichenkette s. Aus "Testfunktion" würde z.B. "tfunktion".
