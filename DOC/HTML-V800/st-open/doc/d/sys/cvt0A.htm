h2str
Funktion
rc = h2str(quelle, ziel, anzahl, format);
Aufruf

DD
format
Abstand der Leerzeichen (0 = unformatiert)
Eingabe
DD
rc
immer Null
Ausgabe
h2str() wandelt [anzahl] Bytes ab Adresse [quelle] in eine formatierte ASCII-Zeichenkette ab Adresse [ziel] um. Nach jeweils [format] Bytes wird ein Leerzeichen zur Formatierung eingefügt. Ist [format] Null, dann erfolgt keine Formatierung. Der Zielspeicher muss grösser als 2 * [anzahl] + Formatierbytes sein!
Beschreibung
{
 DB s[32];
 h2str(s, "0123456789", 10, 4);
}
Die Zeichenkette s enthält nun "3031 3233 3435 3637 3839".
