sys.lib
Fopen
Funktion
rc = Fopen(pfh, grösse, modus, adresse);
Aufruf
PFH
pfh
Adresse einer FH-Struktur
DD
grösse
Grösse (0 = ganze Datei)
DD
modus
00 öffnen oder anlegen
01 anlegen
02 öffnen wenn Datei existiert
03 überschreiben wenn Datei existiert
VOID *
adresse
Adresse des Dateinamens
Eingabe
DD
rc
Fehlercodes von DosOpen(), siehe Quelltext
Ausgabe
Fopen() öffnet die Datei [name] im angegebenen [modus] (Modus 01 erzeugt einen Fehler, wenn die Datei existiert, Modi 02 und 03 erzeugen einen Fehler, wenn die Datei nicht existiert). Die Datei kann nach dem Öffnen über die FH-Struktur angesprochen werden.
Beschreibung
DD MeineFunktion(VOID *name)
{
 FH fh;
 DD rc;
 Mcl(&fh, 24);
 rc = Fopen(&fh, 0, 0, name);
 if(rc)
 {
  return(rc);
 }
 .
 .
}
rc enthält den Fehlercode von DosOpen().
