Ausgabe
asc2s() wandelt [anzahl] Bytes ab Adresse [quelle] in ASCII-Zeichen um, die ab der Adresse [ziel] abgelegt werden. Bytes mit einem Wert unter 0x20 werden als zwei Punkte ".." dargestellt.
Diese Funktion ist zum Beispiel für die ASCII-Spalte von Hexeditoren nützlich.
Beschreibung
{
 DD a = 0x12345678;
 DB s[16];
 asc2s(&a, s, 4);
}
Die Zeichenkette s enthält nun ".4Vx".
