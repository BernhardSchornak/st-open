Q2hex

rc = Q2hex(zahl, ziel);
Aufruf

RAX
rc
immer Null
Ausgabe
Q2hex() wandelt das übergebene Vierfachwort in eine ASCII-Zeichenkette um, die in zwei Gruppen zu je acht Ziffern formatiert ist.
Der übergebene Puffer muss eine Mindestgrösse von 32 Byte haben! (Die zurückgelieferte Zeichenkette hat 15 abschliessende Nullen.)
Beschreibung
{
 DB s[32];
 Q2hex(0123456789ABCDEF, s);
}
Die Zeichenkette s enthält nun "01234567 89ABCDEF".
