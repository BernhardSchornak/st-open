CntWrd
Funktion
rc = CntWrd(ziel);
Aufruf
VOID *
ziel
Adresse der Zeichenkette
Eingabe
DD
rc
Anzahl Worte
Ausgabe
CntWrd() zählt die Worte in der übergebenen Zeichenkette und liefert deren Anzahl zurück. Als Wort wird jede Teilzeichenkette betrachtet, die mit einem Leerzeichen abgeschlossen ist. Die Anzahl der Worte ist demnach die Anzahl der Leerzeichen plus Eins.
Beschreibung
VOID MeineFunktion(VOID)
{
 DB s[32];
 SCpy(s, "Teste diese Zeichenkette");
 rc = CntWrd(s);
}
rc enthält nun den Wert 0x03.
