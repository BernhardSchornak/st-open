Ausgabe
LDfre() gibt einen allozierten Speicherblock frei. Geänderte Datenfelder werden vom Lader automatisch gesichert. Dateien können optional automatisch gesichert werden, siehe Dateien laden.
Da der Speicher dynamisch verwaltet wird, wird bei der nächsten Anforderung der frei gewordene Speicherhenkel wieder verwendet. Daher sollten sollten alle Variablen, die Speicherhenkel oder Adressen von Speicherblöcken enthalten, direkt nach der Freigabe auf Null setzen, um mögliche Fehler zu vermeiden.
Beschreibung
DD MeineFunktion(DD mh)
{
 DD rc;
 rc = LDfre(mh);
 return(rc);
}
