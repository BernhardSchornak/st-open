D2dec
Funktion
rc = D2dec(zahl, ziel, stellen);
Aufruf
DD
zahl
Umzuwandelndes Doppelwort
VOID *
ziel
Adresse des Ziels
DD
stellen
Byte 00: Anzahl Dezimalstellen(0 => 10)
Byte 01: Anzahl Kommastellen
Eingabe
DD
rc
immer Null
Ausgabe
D2dec() wandelt [zahl] in eine in Dreiergruppen formatierte Dezimalzeichenkette ab Adresse [ziel] um. Die Dezimalstellenanzahl wird in Byte 00 des Parameters [stellen] festgelegt. Führende Nullen werden bei der Umwandlung automatisch in Leerzeichen umgewandelt, so dass Kolonnen mit Zahlen unterschiedlicher Grösse bei Verwendung von Festbreitenschriftarten korrekt dargestellt werden.
Byte 01 dieses Parameters legt die Anzahl der Pseudofliesskommastellen fest. Ist die Dezimalstellenanzahl Null, wird sie von D2dec() automatisch auf zehn gesetzt.
ACHTUNG: Die Ausgabezeichenkette muss mindestens 16 Byte gross sein!
Beschreibung
{
 DB s[16];
 D2dec(0x12345678, s, 0x0204);
}
Die Zeichenkette s enthält nun "98.96". Der wirkliche Wert wäre eigentlich "3 054 198.96", die höchstwertigen Stellen werden durch die Beschränkung auf 4 Stellen jedoch abgeschnitten. Sie sollten daher auf den Wertebereich achten oder die Formatierung der Einfachheit halber generell auf 10 Stellen setzen (führende Nullen werden durch Leerzeichen ersetzt).
Übliche Kolonnendarstellung:
1
9999.99
22.44
333.1
Bei Verwendung von D2dec(): 
    1.00
9 999.99
   22.44
  333.10
