dec2Q
Funktion
rc = dec2Q(quelle);
Aufruf
RCX
quelle
Adresse des Quellstrings
Eingabe
RAX
rc
Umgewandelte Zahl
Ausgabe
dec2Q() wandelt die Zeichenkette an der Adresse [quelle] in ein Vierfachwort (8 Byte) um und gibt das Ergebnis zurück. Die Zeichenkette wird rückwärts ausgewertet, überzählige Stellen werden ignoriert. Ein führendes Minuszeichen wird als Vorzeichen erkannt, die Zahl wird ins Zweierkomplement umgesetzt. Beachten Sie, dass das Zweierkomplement von Zahlen grösser 0x7FFFFFFFFFFFFFFF eine positive Zahl ist!
Beschreibung
{
 DB s[16];
 SCpy(s, "10 Gabeln");
 rc = dec2Q(s);
}
Der Rückgabewert rc ist das Vierfachwort 0000 0000 0000 000A, da nur die ersten zwei ASCII-Zeichen gültige Dezimalziffern darstellen.
