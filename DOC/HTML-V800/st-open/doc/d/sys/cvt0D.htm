Q2dec
Funktion
rc = Q2dec(qwort, ziel, format);
Aufruf
VOID *
qwort
Adresse des umzuwandelnden Vierfachwortes
VOID *
ziel
Adresse des Ziels
DD
format
FFFFFFFF = keine Formatierung
00000000 = Formatierung ohne Kommastellen
000000nn = Dreiergruppen mit nn Kommastellen
Eingabe
DD
rc
immer Null
Ausgabe
Q2dec() wandelt [qwort] in eine Zeichenkette ab Adresse [ziel] um. Der dritte Parameter legt fest, ob und wie die Zeichenkette formatiert werden soll. Die Pseudofliesskommastellenanzahl darf nicht grösser als 19 sein!
ACHTUNG: Der Ausgabepuffer muss mindestens 32 Byte gross sein!
ACHTUNG: Das übergebene Vierfachwort wird als vorzeichenbehaftet angesehen. Ist Bit 63 gesetzt, wird automatisch ein Minuszeichen vor die Zahl gesetzt und ihr Zweierkomplement ausgegeben.
Beschreibung
{
 DQ q = 0x1234567890123;
 DB s[32];
 Q2dec(&q, s, 2);
}
Die Zeichenkette s enthält nun "        3 202 559 734 582.11" (8 führende Leerzeichen).
