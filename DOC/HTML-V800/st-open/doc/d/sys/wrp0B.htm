SetPath
Funktion
SetPath(adresse);
Aufruf
DB *
adresse
Adresse des Dateinamens (argv[0])
Eingabe
DD
rc
Immer Null
Ausgabe
SetPath() extrahiert den Pfad und das Laufwerk aus der übergebenen Zeichenkette und setzt das aktuelle Laufwerk und den aktuellen Pfad mit diesen Werten. Diese Funktion wird z.B. benötigt, wenn Sie Dateien öffnen wollen, die auf das Programmicon gezogen oder auf der Kommandozeile übergeben werden.
Beschreibung
DD MeineFunktion(DB *name)
{
 SetPath(name);
}
