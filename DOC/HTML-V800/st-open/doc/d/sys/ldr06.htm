DD

Null
Ausgabe
FnDat() wandelt die übergebene Feldnummer in einen Dateinamen um, der in der übergebenen Zeichenkette abgelegt wird. Feldnummern ab 0xFFFFFFF0 werden in Dateinamen der Form "..\syst\xxx.dat", alle darunter liegenden Nummern in Dateinamen der Form "..\data\xxxxxxxx" umgewandelt.
Beschreibung
VOID MeineFunktion(VOID *name)
{
 FnDat(0x0100, name);
}
Die Ausgabezeichenkette enthält nun "..\data\00000100".
