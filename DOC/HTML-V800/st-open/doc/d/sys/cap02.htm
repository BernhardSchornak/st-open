LdIcon
Sandkasten
hicon = LdIcon(hinst, ikonenname);
oder
hicon = LdIcon(hinst, (LPCTSTR)id);
Aufruf
RCX
hinst
Instanzhenkel der Applikation
RDX
ikonenname
Adresse des Ikonennamens
oder
RDX
id
Resource-ID
Eingabe
RAX
hicon
HICON (0 = Fehler)
Ausgabe
LdIcon() ruft LoadIconA() auf, und stellt anschliessend die von der API überschriebenen Register ECX, EDX, R08, R09, R10, R11, XMM4 und XMM5 wieder her.
Verwenden Sie Sandkästen nicht in Hochsprachen. Mit schlampigen Konventionen arbeitende Kompilierer und API-Funktionen überschreiben die mühsam wiederhergestellten Register sofort wieder mit ihrem gerade anfallenden temporären Datenmüll. Nur saubere Programmierung profitiert von Registern, deren Inhalt auch über Funktionsaufrufe hinweg unverändert bleibt.
Beschreibung
 ...                   # RCX = HINST
 movl  $0x1005, %edx   # RDX = ID icon
 call  _LdIcon
 movq  %rax,    %rbp   # RBP = Hicon
 ...
Ermittelt den Ikonenhenkel der Applikationsikone. Die Resource-ID ist hier in einer Resourcedatei mit der Endung ".rc" definiert.
