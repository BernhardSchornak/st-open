Ausgabe
WnPos() ruft WinSetWindowPos() auf und stellt anschliessend den Inhalt der vom API zerstörten Register ECX und EDX wieder her.
Rufen Sie diesen Sandkasten nicht von C aus auf. Schlampige Programmierumgebungen profitieren nicht von wiederhergestellten Registern. Sie verschwenden etliche Taktzyklen für das unnötige Wiederherstellen von Registerinhalten, die sofort wieder mit aus dem Speicher geladenen Parametern überschrieben werden.
Beschreibung
{
 WnPos(hwnd, HWND_TOP, 100, 100, 200, 200, SWP_SIZE | SWP_MOVE | SWP_SHOW);
}
Zeichnet das Fenster mit dem Henkel hwnd über alle anderen Fenster (HWND_TOP). Die linke untere Ecke ist 100 Pixel horizontal und 100 Pixel vertikal von der linken unteren Bildschirmkante (Position 0, 0) entfernt, das Fenster hat eine Grösse von 200 x 200 Pixel.
