RegClass
Sandkasten
id = RegClass(wndclass);
Aufruf
RCX
wndclass
Adresse einer WNDCLASS-Struktur
Eingabe
RAX
id
eindeutige Klassen-ID
Ausgabe
RegClass() ruft RegisterClassA() auf und stellt anschliessend die von der überschriebenen Register ECX, EDX, R08, R09, R10, R11, XMM4 und XMM5 wieder her.
Mit RegClass() kann die kleine Ikone nicht gesetzt werden. Verwenden Sie RegClssX(), um auch die kleine Ikone setzen zu können.
Verwenden Sie Sandkästen nicht in Hochsprachen. Mit schlampigen Konventionen arbeitende Kompilierer und API-Funktionen überschreiben die mühsam wiederhergestellten Register sofort wieder mit ihrem gerade anfallenden temporären Datenmüll. Nur saubere Programmierung profitiert von Registern, deren Inhalt auch über Funktionsaufrufe hinweg unverändert bleibt.
Beschreibung
 ...
 pxor  %xmm0,              %xmm0        # XM0 = 0
 leaq  _MainWndProc(%rip), %r9          # R09 = EA Fensterprozedur
 leaq  0x60(%rsp),         %rcx         # RCX = EA WC
 leaq  LC00(%rip),         %rdx         # RDX = EA Menuename
 leaq  LC01(%rip),         %r8          # R08 = EA Klassenname
 movq  %xmm0,              0x60(%rsp)   # 0x00[WC] = 0
 movq  %r9,                0x68(%rsp)   # 0x08[WC] = EA Fensterprozedur
 movq  %xmm0,              0x70(%rsp)   # 0x10[WC] = 0, 0x14[WC] = 0
 movq  %rsi,               0x78(%rsp)   # 0x18[WC] = Hinstance
 movq  %rbp,               0x80(%rsp)   # 0x20[WC] = Hicon
 movq  %rdi,               0x88(%rsp)   # 0x28[WC] = Hcursor
 movq  %rax,               0x90(%rsp)   # 0x30[WC] = Hintergrundfarbe
 movq  %rdx,               0x98(%rsp)   # 0x38[WC] = EA Menuename
 movq  %r8,                0xA0(%rsp)   # 0x40[WC] = EA Klassenname
 leaq  0x60(%rsp),         %rcx         # RCX = EA WC
 call  _RegClass
 testl %eax,               %eax         # registriert?
 je    L00
 ...
LC00 und LC01 sind in .rdata definiert, WC steht für WNDCLASS.
