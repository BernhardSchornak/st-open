prime
Funktion
rc = prime(zahl);
Aufruf
DD
zahl
Zu prüfende Zahl
Eingabe
DD
rc
1 = Primzahl, 0 = keine Primzahl
Ausgabe
prime() prüft, ob die übergebene Zahl eine Primzahl ist. Das Ergebnis ist 1, wenn eine Primzahl vorliegt, ansonsten wird eine Null zurückgegeben.
Beschreibung
{
 DD rc;
 rc = prime(13);
}
rc liefert 1 zuück, da 13 eine Primzahl ist.
