RAND

Zufallszahl
Ausgabe
RAND() erzeugt eine Zufallszahl zwischen 0 und 4.294.967.295.
RAND() verwendet das Performanceregister der CPU zur Erzeugung einer Zufallszahl. RAND() benötigt zwischen 30 und 80 Taktzyklen (abhängig vom Inhalt des Zählers), um eine Zufallszahl zu erzeugen. Da der Zähler mit dem CPU-Takt läuft, ist die Qualität der erzeugten Zufallszahl so hoch, dass sie jederzeit zur Verschlüsselung oder für andere kritische Aufgaben verwendet werden kann.
Beschreibung
DD MeineFunktion(VOID)
{
 DD rc;
 rc = RAND();
 return(rc);
}
rc enthält nun eine Zufallszahl zwischen 0x00000000 und 0xFFFFFFFF.
