Diese Funktion ist Teil der in das System integrierten Sprachunterstützung.

MNUtxt

Fensterhenkel (Klient)

Eintragsnummer des ersten Menütexts
DD

Eintragsnummer des letzten Menütexts
Eingabe
DD
rc
ERR_NO_ERROR

Ausgabe
MNUtxt() schreibt den Inhalt des durch STV_USR_LANGUAGE festgelegten Unterfeldes des Feldes FFFFFF12 - beginnend mit dem Eintrag [anfang] bis einschliesslich dem Eintrag [ende] - in das Menü des Elternfensters von [hwnd]. Die Resource-ID des jeweiligen Menüeintrags wird aus dem Unterfeld 16 des Feldes FFFFFF12 gelesen.
Beschreibung
DD MeineFunktion(HWND hwnd)
{
 *STV_USR_LANGUAGE = 0;
 MNUtxt(hwnd, 0, 33);
}
Die Sprache wird auf Englisch (0) gesetzt. Anschliessend werden die Menüeinträge 0 bis 33 mit den in Unterfeld 16 festgelegten Resource-IDs mit den aus Unterfeld 0 gelesenen Texten überschrieben.
