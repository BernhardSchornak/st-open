Output
FileOpen() öffnet OS/2's "Datei Öffnen"-Dialog. Der ausgewählte filename wird in die Systemzeichenkette STR_FLS_SELECT kopiert. Wurde keine Auswahl getroffen, bzw. ein Fehler ist aufgetreten, so wird STR_FLS_SELECT gelöscht.
Ist die Systemvariable STV_FLS_ADD ungleich Null, so wird der filename an die Spitze der Dateiliste geschrieben. Alle anderen Einträge der Dateiliste werden hierbei nach unten verschoben. Enthält die Dateiliste bereits 256 filenamen, so wird entry 255 gelöscht.
Description
DD MyFunction(HWND hwnd) 
{
 DD rc;
 FDacc(FLD_STR, STR_FLS_SELECT, 0, FDA_WRITE, "*.*", 0);
 rc = FileOpen(hwnd);
 return(rc);
}
STR_FLS_SELECT wird auf das Suchmuster gesetzt und der Dialog wird geöffnet. Wenn eine Datei ausgewählt wurde, enthält STR_FLS_SELECT den filenamen, ansonsten eine leere string. Ist die Systemvariable STV_FLS_ADD ungleich Null, so wird der filename zusätzlich an die Dateiliste angehängt.
