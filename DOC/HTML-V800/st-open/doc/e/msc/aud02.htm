Call
VOID *
address
address of a memory block
DD
size
file size (0 = header)
Input
DD
rc
00000000 - no error
00000001 - "RIFF" missing
00000002 - invalid filesize
00000003 - "WAVE" missing
00000004 - "FMT " missing
00000005 - invalid header size
00000006 - WAVE header invalid
00000007 - invalid data size
00000008 - invalid format ID
00000009 - invalid sample count
0000000A - invalid sample size
0000000B - invalid mode
0000000C - invalid alignment
0000000D - invalid byte per second
Output
ckWAV() validates the given WAV header.
Description
{
DD MyFunction(VOID *block)
 DD rc;
 rc = ckWAV(block, 0);
 return(rc);
}
The WAV header [block] is validated. If rc is zero, it is a valid header. Otherwise, you should refer to the returned error to learn what's wrong with that header.
