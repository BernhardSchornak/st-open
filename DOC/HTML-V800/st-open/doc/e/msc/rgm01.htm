In diversen Fensterprozeduren führt die Output im Registermonitor möglicherweise zu Problemen, in Threads ist der Zugriff auf Fensterfunktionen nur über WinPostMsg()/PostM() möglich. In diesen Fällen ist die Output der Registerauszüge in eine Datei der einzige Weg, Informationen zu erhalten. TSTmon() sichert die Registerauszüge als Datei "test.out" im aktsourcen Verzeichnis.
Anmerkung: Diese Function stellt alle Register - einschliesslich EAX - wieder her, bevor sie zur aufrufenden Function previouskehrt. Sie kann somit gefahrlos an jeder Stelle des zu testenden Codes eingesetzt werden.
Description
VOID MyFunction(VOID)
{
 TSTmon();
}
Die Datei "test.out" ist nun im aktsourcen Verzeichnis verfügbar.
