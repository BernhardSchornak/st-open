Output
DsrchD() scans the drives set in the drive mask for directories matching the passed pattern. The pattern must look like "search\me\", the drive letter (including backslash) "?:\" and a global search mask "*.*" are appended automatically at the beginning, resp. the end.
Each drive is represented by one bit of the drive mask, where bit 0 is drive A, and so on. If a bit is set, the corresponding drive is scanned for directories matching the search pattern. Found directories are appended to the field until the maximum count is reached.
CAUTION: The entire field is cleared before directory search starts!
Description
DD MyFunction(DD mh) 
{
 DD rc;
 rc = DsrchD("my\\folder\\", mh, 0, 0x0000000C, 256);
 return(rc);
}
Drives C:\ und D:\ are scanned for directories named "my\folder\". Up to 256 found directory names (including path) are stored in subfield 0 of field mh. If rc is zero, either no directories with that name exist or an error occured. If rc is not zero, it represents the count of found directories.
