FsrchP() durchsucht das Dateisystem nach dem übergebenen Muster. Bis zu [annumber] gefundene filenamen (ohne Pfad) werden im angegebenen subfield des angegebenen fieldes gespeichert.
ACHTUNG: Ist [start] Null, so wird das gesamte field wird vor dem Suchen gelöscht!
Description
DD MyFunction(DD mh) 
{
 DD rc;
 rc = FsrchP("*.ego", mh, 0, 0, 256);
 return(rc);
}
Das aktsource Verzeichnis wird nach Dateien mit der Endung "ego" durchsucht. Da [start] Null ist, wird das field vor der Suche gelöscht. Bis zu 256 gefundene filenamen (mit Pfad) werden im field [mh], subfield 0 gespeichert. Der Rückgabewert rc enthält die amount gefundener Dateien. Ist er Null, dann wurde kein Verzeichnis gefunden, bzw. ein Fehler führte zum Abbruch der Function.
