LoadDLL() loads the dynamic link library with the name stored at [address]. Accessing a function in the DLL is done via the ordinal number of that function (see sourcecode of PlayFile()).
Description
DD MyFunction(VOID *name)
{
 DD rc;
 rc = LoadDLL(name);
 return(rc);
}
Returns the modul handle of the DLL. If rc is NULL, the DLL couldn't be loaded.
