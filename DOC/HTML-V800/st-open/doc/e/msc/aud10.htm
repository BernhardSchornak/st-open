mix16() mixes up to 32 sources to one target WAV file. Due to the simple design of this function, the amount of source files must be 2, 4, 8, 16 or 32. The format of all source files must be the same - mixed formats are rejected.
Description
DD MyFunction(VOID *array, VOID *target)
{
 DD rc;
 rc = mix16(array, target, 8, 0x2000);
 return(rc);
}
Mixes the 8 source files passed in [array] and stores the result (8,192 byte) in [target].
