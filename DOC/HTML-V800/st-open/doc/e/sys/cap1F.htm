Output
WInvR() calls WinInvalidateRect(), then restores the content of ECX and EDX because both are destroyed by the API.
Please do not call this sandbox from C. Dirty programming environments do not benefit from properly restored registers and you waste a lot of clock cycles to preserve registers which are filled with garbage again soon after returning.
Description
{
 WInvR(hwnd, 0, 0);
}
The rectangle of window hwnd is added to the list of invalid screen areas and a WM_PAINT message is sent to the window procedure associated with hwnd.
