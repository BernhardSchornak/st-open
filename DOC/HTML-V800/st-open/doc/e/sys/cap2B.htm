MBox() calls MessageBoxA(), then restores the destroyed content of RCX, RDX, R08, R09, R10, R11, XMM4 and XMM5.
Please do not call sandboxes from C code. It is a waste of precious time to restore registers which are refilled with random garbage soon after.
Description
{
 char s[16] = "Question",
      t[16] = "Your choice:";
 long rc;
 ...
 rc = MBox(s, t, MB_YESNO | MB_ICONQUESTION);
 ...
}
Creates a message box with title "Question" and text "Your choice:". The first flag creates two buttons "Yes" and "No", the second flag adds a question mark icon in the upper left corner of the box. Depending on the user's choice, rc is 1 (pressed "Yes") or 2 (pressed "No").
