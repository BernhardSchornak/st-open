Output
LDunc() setzt das "geändert"-Fähnchen des fieldes previous.
ACHTUNG: Änderungen in einem Datenfeld werden nicht gesichert, wenn das Datenfeld direkt nach dieser Operation geschlossen wird!
Description
DD Meinefunction(DD mh)
{
 LDunc(mh);
 LDfre(mh);
 return;
}
Die function löscht das "geändert"-Fähnchen und schliesst dann das field. Da das "geändert"-Fähnchen nicht gesetzt ist, gibt der Lader den Speicher ohne Sichern des fieldes frei. Diese function ist nützlich, wenn Sie vorgenommene Änderungen an einem field verwerfen wollen. Sie sollten das field anschliessend freigeben, da die Datenbankmaschine das "geändert"-Fähnchen bei der nächsten Schreiboperation wieder setzt.
