Output
ThrdSwi2() calls SwitchToThread(), then restores the destroyed content of RCX, RDX, R08, R09, R10, R11, XMM4 and XMM5.
Please do not call sandboxes from C code. It is a waste of precious time to restore registers which are refilled with random garbage soon after.
Description
See your WinAPI documentation how to use this function. (To improve the speed of ST-Open applications, threading was removed long time ago.)
