FreMem() is a low level function. You surely want to use ST-Open's Loader...
Please do not call this sandbox from C. Dirty programming environments do not benefit from properly restored registers and you waste a lot of clock cycles to preserve registers which are filled with garbage again soon after returning.
Desription
{
 VOID *ea;
 DD    rc, sz = 0x1000;
 ...
 rc = AloMem(ea, sz);
 if(rc)
 {
  /* error handling */
 }
 ...
 rc = FreMem(ea, sz);
 ...
}
Frees the allocated memory block at address ea.
