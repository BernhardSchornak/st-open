Diese function ist Teil der in das System integrierten Sprachunterstützung.

Output
DLGtxt() schreibt den Inhalt des durch STV_USR_LANGUAGE festgelegten subfields des fieldes FFFFFF12 - beginnend mit dem entry [anfang] bis einschliesslich dem entry [ende] - in den Dialog [hwnd]. Die Resource-ID des jeweiligen Dialogelements wird aus subfield 16 des fieldes FFFFFF12 gelesen.
Der entry [titel] muss der Text für das Fenster sein, das durch [hwnd] spezifiziert ist. Dieser entry muss einen Text enthalten. Sind [anfang] und [ende] gleich Null oder identisch mit [titel], so wird nur der Fenstertitel gesetzt. Für den entry [titel] wird keine Resource-ID benötigt, da dieser Text direkt an das Fenster [hwnd] gesendet wird.
Description
DD Meinefunction(HWND hwnd)
{
 PUT(STV_USR_LANGUAGE, 1);
 DLGtxt(hwnd, 200, 200, 200);
 DLGtxt(hwnd, 200, 201, 205);
}
Die Sprache wird auf Deutsch (subfield 1) gesetzt. Der erste Call setzt nur den Text des Fensters [hwnd], der zweite sowohl den Fenstertext als auch die Texte von fünf weiteren Texten im Dialog [hwnd], deren Resource-IDs aus subfield 16 des fieldes FFFFFF12 gelesen werden.
