Q2dec
Function
rc = Q2dec(qword, target, format);
Call
VOID *
qword
address where the qword is stored
VOID *
target
address target buffer
DD
format
0 = no formatting
1 = groups of 3 digits
2 = groups of 3 digits and 2 pseudo FP digits
Input
DD
rc
always zero
Output
Q2dec() converts [qwort] into a string in decimal notation. The output is stored in [target]. The third parameter determines the format of the string. To display other pseudo FP formats choose 0 (no formatting) and call dec2f() with the desired amount of FP digits.
Caution: The target buffer must have a size of at least 32 byte!
Description
{
 DQ q = 0x1234567890123;
 DB s[32];
 Q2dec(&q, s, 2);
}
String s now contains "     3 202 559 734 582.11" (5 leading blanks).
