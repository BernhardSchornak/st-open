ErrMgr() gibt Programm- und Systemnachrichten in einer Dialogbox mit bis zu drei Auswahlknöpfen aus. Eine ausführliche Description finden Sie im Tutorial Sprachunterstützung.
Description
DD Meinefunction()
{
 DD rc = ErrMgr(33);
 if(rc == 3)
 {
  /* mach was */
 }
 else
 {
  /* mach was anderes */
 }
}
Gibt die Nachricht mit der Nummer 33 aus. Der Rückgabewert ist die Nummer des vom Benutzer gedrückten Knopfes, die hier zur Verzweigung ausgewertet wird.
