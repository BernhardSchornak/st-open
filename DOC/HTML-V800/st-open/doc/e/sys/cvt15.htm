dec2Q
Function
rc = dec2Q(source);
Call
VOID *
source
address source buffer
Input
DD
rc
always zero
Output
dec2Q() converts the decimal string found in [source] into a qword (8 byte) and returns the result in [source]. The string is evaluated backwards, surplus digits are ignored. A leading minus is recognised as sign and the qword is turned into its two's complement before it is returned.
Note: The two's complement of numbers above 9,223,372,036,854,775,807 is a positive number!
Description
{
 DB s[16];
 SCpy(s, "10 Bears");
 dec2Q(s);
}
String s now contains the qword [0A 00 00 00 00 00 00 00] (only 0 and 1 are valid decimal digits).
