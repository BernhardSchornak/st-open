Output
s2num() converts [amount] ASCII chars found in [source] into unpacked numeric digits of the given base. The result is stored in [target]. Valid chars are 0x30...0x39 (base 10 and 16) and 0x41...0x46 or 0x61...0x66 Hex (base 16, only). All other chars are rejected because they are no numbers. s2num() returns the amount of converted digits.
Description
{
 DD rc;
 DB s[16], t[16];
 SCpy(t, "1c3krt4s1b");
 rc = s2num(t, s, 10, 16);
}
String s now contains "01 0C 03 04 01 0B 00", the returned amount of digits is 6.
