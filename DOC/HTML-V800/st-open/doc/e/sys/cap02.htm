LdIcon
Function
rc = LdIcon(hinst, iconname);
or
rc = LdIcon(hinst, (LPCTSTR)id);
Call
RCX
hinst
instance handle of the application
RDX
iconname
address where iconname is stored
or
RDX
id
resource ID
Input
RAX
hicon
HICON (0 = error)
Output
LdIcon() calls Load IconA(), then restores the destroyed content of ECX, EDX, R08, R09, R10 and R11.
Please do not call sandboxes from C code. It is a waste of precious time to restore registers which are refilled with random garbage soon after.
Description
 ...                                          # RCX = HINST
 movl  $0x1005, %edx                          # RDX = ID icon
 call  _LdIcon
 movq  %rax, %rbp                             # RBP = Hicon
 ...
Retrieves an icon handle HICON for the application specified by HINST.
