Q2str
Function
rc = Q2str(source, target);
Call
VOID *
source
address where the qword is stored
VOID *
target
address target buffer
Input
DD
rc
always zero
Output
Q2str() converts the qword found at address [source] into an ASCII string. The output is formatted in four groups with 4 digits, each.
Description
{
 DB s[16];
 Q2str(s, "12345678");
}
String s now contains "3132 3334 3536 3738".
