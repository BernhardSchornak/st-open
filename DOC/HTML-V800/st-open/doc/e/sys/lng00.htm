Diese function ist Teil der in das System integrierten Sprachunterstützung.
..\st7\lib\sources\core\lng.S
sys.lib
MNUtxt
Function
rc = MNUtxt(hwnd, anfang, ende);
Call
HWND

window handle (Klient)
DD
anfang
entrysnummer des ersten Menütexts
DD
ende
entrysnummer des letzten Menütexts
Input
DD
rc
ERR_NO_ERROR
ERR_LDR_ALLOC
Output
MNUtxt() schreibt den Inhalt des durch STV_USR_LANGUAGE festgelegten subfieldes des fieldes FFFFFF12 - beginnend mit dem entry [anfang] bis einschliesslich dem entry [ende] - in das Menü des Elternfensters von [hwnd]. Die Resource-ID des jeweiligen Menüeintrags wird aus dem subfield 16 des fieldes FFFFFF12 gelesen.
Description
DD Meinefunction(HWND hwnd)
{
 *STV_USR_LANGUAGE = 0;
 MNUtxt(hwnd, 0, 33);
}
Die Sprache wird auf Englisch (0) gesetzt. Anschliessend werden die Menüeinträge 0 bis 33 mit den in subfield 16 festgelegten Resource-IDs mit den aus subfield 0 gelesenen Texten überschrieben.
