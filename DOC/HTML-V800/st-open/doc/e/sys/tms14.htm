mkDate() setzt aus den übergebenen Parametern ein gültiges Datum zusammen. Eine Überprüfung der Parameter erfolgt vor dem Zusammensetzen der einzelnen Teile.
Description
DD Meinefunction(VOID)
{
 rc = mkDate(33, 14, 2005);
 return(rc);
}
rc enthält nun den Wert 0x07D60305 (der 33.14.2005 wurde auf das gültige Datum 05.03.2006 korrigiert).
