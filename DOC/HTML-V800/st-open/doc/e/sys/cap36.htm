Output
GetRes() calls SystemParametersInfoA(), then restores the destroyed content of RCX, RDX, R08, R09, R10, R11, XMM4 and XMM5.
Width and height of the screen are stored as global variables STV_SCREEN_W and STV_SCREEN_H.
Description
{
 GetRes();
}
STV_SCREEN_W and STV_SCREEN_H now contain the current screen resolution.
