tDiff() berechnet die Differenz zwischen den beiden übergebenen Zeiten und liefert das Ergebnis previous. Das Ergebnis ist keine gültige Zeitangabe!
Description
DD Meinefunction(VOID)

    zeit2 = 0x00000000, /* 00:00:00.00 */
    rc;
 rc = tDiff(zeit1, zeit2);
 return(rc);
}
rc enthält nun 0x173B3B63, das sind 23 Stunden, 59 Minuten, 59 Sekunden und 99 Hundertstelsekunden.
