Output
d2out() wandelt das übergebene Datum in eine formatierte string ab address [adresse] um. Das in Europa übliche Zeitformat der Form DD.MM.YYYY wird ausgegeben.
Description
DD Meinefunction(VOID)
{
 DB s[32];
 d2out(0x07D60C12, s);
}
s enthält nun "18.12.2006".
