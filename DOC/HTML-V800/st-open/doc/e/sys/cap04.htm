GStkObj
Function
rc = GStkObj(objtype);
Call
RCX
objtype
predefined stock object ID
Input
RAX
HGDIOBJ
handle of the requested stock object
Output
GStkObj() calls GetStockObject(), then restores the destroyed content of ECX, EDX, R08, R09, R10 and R11.
Please do not call sandboxes from C code. It is a waste of precious time to restore registers which are refilled with random garbage soon after.
