Output
dec2f() converts the string passed in source into a formatted pseudo floating point number. The amount of floating point digits is determined by parameter 2. Not present digits behind the floating point are filled with zeroes, leading zeroes in the integer part are suppressed by replacing them with blanks.
dec2f() can handle strings of any size, so your source buffer must be of sufficient size to hold the entire output without overflow!
Note: This is not a floating point format! We just 'implant' a floating point at a fixed position within an integer.
Description
{
 DB s[16];
 SCpy(s, "1 234 567 890");
 dec2f(s, 5);
}
String s now contains " 12 345.67890".
