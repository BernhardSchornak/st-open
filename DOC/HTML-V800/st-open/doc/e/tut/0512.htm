FDA_SEARCH


Search for the [numeric value] or the pattern passed in [address] in [MemHandle][sub-field], starting with entry number [entry]. If the value or pattern is found, the entry number of the first matching entry is returned. 0xFFFFFFFF (-1) signals "no match".

Numeric Data

bytes, Words und Dwords

DD My_Function(DD mh)
{
 DD a;
 a = FDacc(mh, 0, 0, FDA_SEARCH, 0, 0x12345678);
 return(a);
}

Qwords

DD My_Function(DD mh)
{
 DQ q;
 DD a;
 q = 0x0123456789ABCDEF;
 a = FDacc(mh, 0, 0, FDA_SEARCH, &q, 0);
 return(a);
}

Strings

DD My_Function(DD mh)
{
 DD a;
 DB s[32];
 SCpy(s, "Search for me!");
 a = FDacc(mh, 0, 0, FDA_SEARCH, s, 0);
 return(a);
}

Intelligent Design

You do not need to pass parameter 5 if no address is required. Omit parameter 6 if no numeric value is to pass.
