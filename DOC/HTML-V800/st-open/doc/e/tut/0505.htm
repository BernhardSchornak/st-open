FDA_READ

Read the [entry] stored in [MemHandle][sub-field] and copy it to [address] or return it to the calling function.

Numeric Data

bytes, Words and Dwords

DD My_Function(DD mh)
{
 DD a;
 a = FDacc(mh, 0, 0, FDA_READ, 0, 0);
 return(a);
}

Qwords

DQ My_Function(DD mh)
{
 DQ q;
 FDacc(mh, 0, 0, FDA_READ, &q, 0);
 return(q);
}

Strings

DB * My_Function(DD mh)
{
 DB s[32];
 FDacc(mh, 0, 1, FDA_READ, s, 0);
 return(s);
}

Intelligent Design

You do not need to pass parameter 5 if no address is required. Parameter 6 is not required at all.
