FA

   typedef struct _FA{
                      DD    MMoff;   /* 00   MemHandle             */
                      DD    Field;   /* 04   ignored!              */
                      DD    SubFd;   /* 08   sub field number      */
                      DD    Entry;   /* 0C   entry number          */
                      VOID *StrEA;   /* 10   string adress to pass */
                      DD    Value;   /* 14   value to pass         */
                     } FA;
   typedef FA *PFA;
Definition
MemHandle
MMoff
parameter ignored - please pass a MemHandle in FA.MMoff!
Field
number of the sub-field
SubFd
entry number
Entry
optional: address of a string
StrEA
optional: numeric value
Value
The FA structure is used by a few PM functions. Please refer to those functions. This structure is obsolete and disappears sooner or later. Please replace code using it with direct calls to the appropriate API functions.
