FDA_*


The following pages contain examples how to use DM's function distributor FDacc(). The syntax of any call to FDacc() is:
FDacc([MemHandle], [entry], [sub-field], FDA_COMMAND, [address], [numeric value]);
The parameters "address" and "numeric value" are optional and depend on the chosen command. The terms in square brackets are used in the descriptions as reference to the above definition.

FDA_TEST

FDA_TEST is not a real function, it is just an exit after the parameter checks are done. You may use it to check if the passed parameters are okay. 
DD My_Function(DD mh)
{
 DD rc;
 DB s[32];
 rc = FDacc(mh, 0, 0, FDA_TEST, s, 0);
 return(rc);
}
