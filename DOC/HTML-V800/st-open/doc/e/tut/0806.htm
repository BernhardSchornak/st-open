SecSys beenden

Um das Sicherheitssystem zu beenden, muss nur der Befehl SS_SDN ausgeführt werden: 
SecSy(SS_SDN, 0, 0);
Mit diesem Befehl wird das Sicherheitssystem beendet. Die vom Sicherheitssystem allozierten Speicherbereiche werden freigegeben, temporäre Puffer und Variablen werden mit Nullen gefüllt. Die Kontroll-Logik der Applikation wird durch diesen Befehl in den Status Gast zurückversetzt - es können nur noch solche Aktionen ausgeführt werden, die einem Gast zugestanden werden. Es ist daher angeraten, SS_SDN im Epilog der Applikation auszuführen, um dem Benutzer das Abschalten selbiger zu ermöglichen.
