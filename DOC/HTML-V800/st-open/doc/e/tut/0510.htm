FDA_XCHANGE


exchange the content of [entry] in [MemHandle][sub-field] with the content of entry [numeric value]. This call is identical for all data types. 
VOID My_Function(DD mh)
{
 FDacc(mh, 0, 0, FDA_XCHANGE, 0, 1);
}

Intelligent Design

Parameter 5 is not required.
