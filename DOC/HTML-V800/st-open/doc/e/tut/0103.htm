LDR_*

Loader aktive, access denied
LDR_BUSY
field possibly corrupted
LDR_CORRUPT
field controlled by SecSys
LDR_SECSYS
field content encrypted
LDR_ENCRYPT
write access running, data might be inconsistent (garbage collection)
LDR_DIRTY
reserved
LDR_RES0A
reserved
LDR_RES09
reserved
LDR_RES08
file is not a field
LDR_NO_FIELD
add to loaded files list (LFL)
LDR_LFLIST
set CHK_REMOVE to infinity
LDR_STATIC
memory block, no file loading required
LDR_MEM
don't close file after copying its content to memory
LDR_LEAVE_OPEN
content changed
LDR_CHANGED
write access allowed
LDR_WRITE
read access allowed
LDR_READ
load file (LDR_READ | LDR_WRITE | LDR_STATIC | LDR_NO_FIELD)
LDR_FILE
add file to LFL (LDR_READ | LDR_WRITE | LDR_STATIC | LDR_LFLIST | LDR_NO_FIELD)
LDR_AUTOSAVE
