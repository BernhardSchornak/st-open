FDA_INC


Increment [entry] in [MemHandle][sub-field]. FDA_INC checks if the result is zero. If so, the entry is not touched. In all other cases the entry is overwritten with the new value. The result is returned for data types 01, 02 and 03, data type 04 returns zero by default.
The command FDA_INC is ignored for fixed length data and strings.

bytes, Words und Dwords

DD My_Function(DD mh)
{
 DD a;
 a = FDacc(mh, 0, 0, FDA_INC, 0, 0);
 return(a);
}

Qwords

DQ My_Function(DD mh)
{
 DQ q;
 FDacc(mh, 0, 0, FDA_INC, &q, 0);
 return(q);
}

Intelligent Design

You do not need to pass parameter 5 if no address is required. Parameter 6 is not required at all.
