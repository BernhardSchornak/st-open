Language Support


One of the new features introduced with Version 6 is its field based language support for up to 8 languages. The user can switch from one language to another in the running program and all menu, window and dialog texts are changed to the new language immediately. Because the resource (RC) and dialog (DLG) files don't need any text parts, the size of the program's resource file (RES) - and therefore the size of the program - is reduced. The writing of the texts into the windows is automated and doesn't take much longer than the usual method to load them from the resources linked with the program (which only provide one language).
Entering text, translation and management of the language fields is comfortably done with dialogs especially designed for this task. These dialogs are included in DatTools, ST-Open's toolbox for the creation and editing of data fields. Because DatTools are freely available, potentially every user - even without any programming skills - is able to translate the application to another language (if it works with the provided language).
