The Database Engine

ST-Open's database engine is the largest function in ST-Open's libraries. It can handle 4,294,967,295 fields with a size of up to 4,294,967,295 byte, each. In real life, the Loader provides 256 MemHandles (expandable to multiples of 256) and the OS will not allow to allocate memory beyond the physical limits of your machine.
This tutorial gives an overwiew how fields are organised, explains how ST-Open's DBE works and offers code samples for all DBE subfunctions.
