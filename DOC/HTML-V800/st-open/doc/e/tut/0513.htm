FDA_COMPARE


Compare [entry] in [MemHandle][sub-field] with [numeric value] or the pattern passed in [address]. The returned result is zero if the pattern is equal to the entry, 1 if the entry is greater than the pattern and 2 if the pattern is greater than the entry.

Numeric Data

bytes, Words und Dwords

DD My_Function(DD mh)
{
 DD a;
 a = FDacc(mh, 0, 0, FDA_COMPARE, 0, 0x12345678);
 return(a);
}

Qwords

DD My_Function(DD mh)
{
 DQ q;
 DD a;
 q = 0x0123456789ABCDEF;
 a = FDacc(mh, 0, 0, FDA_COMPARE, &q, 0);
 return(a);
}

Strings

DD My_Function(DD mh)
{
 DD a;
 DB s[32];
 SCpy(s, "Compare me!");
 a = FDacc(mh, 0, 0, FDA_COMPARE, s, 0);
 return(a);
}

Intelligent Design

You do not need to pass parameter 5 if no address is required. Omit parameter 6 if no numeric value is to pass.
