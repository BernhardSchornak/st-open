V800 Program Skeleton

This is a skeleton you can use as a template for new V800 programs. The skeleton creates the frame and client window, then invokes the ST-system. Several samples are included to introduce you to the most used functions and dialogs supplied by ST-Open's libraries.
If you want to write a program using ST-Open's libraries:
1. Copy the V800 skeleton to a folder of your choice.
2. Customize the frame and client window.
3. Add your dialogs.
4. Add your functions and the program logic.
The proper initialization and termination functions for the Loader are included in the skeleton code. You don't have to worry about this stuff and can start writing your own code immediately.
