Programmer's Calculator

The simple 32 bit calculator meanwhile grew to the Programmer's ToolBox, short PTB. It now includes viewers for register, memory and stack dumps which were created by the debugging tools provided in ST-Open's core library. The 32 bit calculator now works with 64 bit registers and a lot of useful gadgets were added. A factor generator for replacing divisions by multiplications is planned, but not coded yet. It is quite tricky to convert existing 32 bit solutions to 64 bit...
