HDrecord

This program is on the development list.
ST's HDrecord is planned as a software replacement for my current equipment, the soundcard mixer. It shall feed an analog mixer with the output signal (multiple channels) from a soundcard, add an instrument to the analog stream, and send the instrument back to the line input of the soundcard. While I currently use some kind of modern overdubbing, where the tape recorder is replaced by a harddisk, HDrecord should work like a real studio, where a PC replaces the mixer as well as the recording machine. Runtime mixing of several tracks while recording is not the best thing to do, because even the best soundcards have an audible delay, caused by quite low performance of the drivers and mixing software, most times coded in C(++). HDrecord creates a pre-mix which is fed to a modified version of the soundcard mixer. It feeds the pre-mix and the instrument to the headphones and the pure instrument back to the soundcard. Because both streams run simultaneously and continuously, there are no latencies.
I hope I find some time to translate my ideas into working code...
