TestTools

This is a collection of tools to test execution times of functions with the RDTSC instruction. For some (weird) technical reasons, this program is included in the V800 archive. It is an internal tool to test runtimes of functions, so it probably is of no use for anyone else than me. Because it is an internal tool, I deny to waste my precious time with writing a documentation. There also is no guarantee of any kind nor any support for any of those tools. Write your own test functions... ;)
